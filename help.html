Review v1.0.6

Review is a customer submitted reviews App that allows you to easily insert a customer review functionality into your websites pages, products and web apps.

Create a Review

  1. Open the BC CMS then navigate to the Customer Reviews App in the left menu and click the icon.
  2. Click + Add a Review App to create a customer submitted review.
  3. Enter the name you want the customer review to be called.
  4. Choose your customer reviews permission. IMPORTANT to enable the Allow Anonymous Reviews feature read the Review Permission section for details.
  5. Use the form builder to add, remove and rearrange the features you want in your review.
  6. Click "Create Review" to create your review.
  7. Insert the customer review into your website using Liquid tags. Refer to ‘Inserting a Review’ below.
  8. and signify an items visibility to users (Click to toggle items visibility).

Edit a Review

  1. Open the BC CMS then navigate to the Customer Review App and click.
  2. Select the review from the menu you wish to edit.
  3. Edit, add, remove and rearrange the review
  4. Click Save.

Review Permissions

  1. Requires Approval - If enabled each review submitted will need to be enabled before it can be viewed on the site.
    Enable Review - Open the Reviews tab in the Customer Reviews Admin app select a review with where the "View" link is red. "View" links in red signify disbaled reviews. Click the "View" link to open the review and check the enable checkbox.
    Disable Review - to disable a review open the review in the admin app with the "View" link and uncheck the enable checkbox.
  2. Allow Anonymous Reviews - If enabled this allows user that are not logged into to add reviews to the site.
    Activate Anonymous Reviews - Navigate to the CRM Search feature in the BC admin and search "Anonymous". Rich Snippets - Structured Data

  • Customer Reviews app uses schema.org structured data to support better search engine visibility.
  • By default Customer Reviews uses the aggregated review schema embedded inside a products schema. If you have a different schema you wish to use then the Customer Review Schema can be found and edited in this directory on your site "/_System/Apps/digifuel-review/public/reviews-schema.tpl" more on schema's can be found here at schema.org or Google's Strutured Data. Test your structured data with Google's Strutured Data Testing Tool.
  • Customer Reviews default structured data - Product name, description & image / Aggregated Reviews ratingCount & ratingValue.

Requirements/Limitations

Requirements: This app requires that JQuery is installed and ‘Page Referrer’ is not disabled in the web browser.

Limitations: Some features of this app are limited by BC Open Admin API’s either by being currently available or limited. As these API’s are made available updates will add features and rectify some of the current limitation.

  • Reviews cannot be renamed
  • The "Requries Aprroval" setting cannot be altered once an app is created.
  • Page referrer must be turned on in the browser

Troubleshooting

I have updated the app from the BC appstore and the app is no longer working.

If you have updated the app to a version 1.0.6 or greater then you will need to first go to each review app and update by clicking save in the edit review tab then replacing the liquid on any page the app is installed with the new liquid tags. You will also need to replace the reviews.js the review app is referencing on any page review is installed or on the template or content holder it was placed to 

<script type="text/javascript" src="/_System/Apps/digifuel-review/public/reviews.js"></script>
.

My customers reviews are not showing in the reviews tab.

Usually this would mean that jQuery and or review.js is not installed correctly on the page the customer review is inserted on. Refer to the section above ‘Inserting a Review’ to make sure all the elements required are on the page.

I cannot rename a review.

Unfortunately you cannot rename the review app in this version. This feature will hopefully be added in a future version as BC Open Admin adds more API’s.

I get an error telling me that ‘Page referrer is blocked’.

This error occurs when the browser you are using has the Page Referrer blocked. To fix this try using another browser or search how to re enable ‘Page Referrer’ in the browser you are using. This requirement will hopefully be lifted as BC Open Admin adds more features.

I have tried everything and the Review App is just not working.

If you have tried everything and the Customer Reviews app is just not working then contact reviewsapp@digitalfuel.co.nz for assistance.

Uninstall App

NOTE: UNINSTALL will delete all the Customer Reviews app settings. However it will not delete the webapps that contain the database of reviews and their information.

WARNING: DELETE will delete all the Customer Reviews app settings and all the webapps that contain the database of reviews and their information. This process cannot be undone.
